Project Overview

LeafGuard AI is a desktop web application that uses a custom Convolutional Neural Network (CNN) to classify tomato plant leaves into four health categories in real time. It was developed to address the need for fast, affordable, and offline-capable disease diagnostics for smallholder farmers ‚Äî particularly in regions like Ghana where crop losses of 20‚Äì40% are common due to undetected disease.

Dataset

Training data sourced from the PlantVillage public dataset (Penn State University). Four tomato-specific subsets were used, with an 80/20 train/validation split and data augmentation (rotation, flips, brightness, zoom).

Technical Stack

  • ML: TensorFlow 2.x / Keras, NumPy, Pillow
  • Frontend: HTML5, CSS3, Vanilla JS (WebRTC for webcam)
  • Training: Google Colab (free GPU)
